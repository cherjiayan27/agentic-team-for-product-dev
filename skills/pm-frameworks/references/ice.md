# ICE Scoring

**When relevant:** Quick feature comparison. Early-stage exploration. When RICE is overkill.

---

## Formula

```
ICE Score = (Impact + Confidence + Ease) / 3
```

## Components

| Component | Scale | Description |
|-----------|-------|-------------|
| **Impact** | 1-10 | Expected effect on key metric |
| **Confidence** | 1-10 | How sure are you about impact? |
| **Ease** | 1-10 | How easy to implement? |

## When to Use ICE vs RICE

| Situation | Use |
|---|---|
| Early-stage exploration, quick estimates | **ICE** |
| Quarterly planning, cross-team prioritization | **RICE** |

ICE is faster and simpler. RICE is more rigorous and accounts for reach.
